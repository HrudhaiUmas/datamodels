Apache Cassandra is a NoSQL columns data model based database management system and the official website can be found at: [Cassandra](https://cassandra.apache.org/_/index.html). Cassandra DB is an open-source NoSQL database. Cassandra has multiple hosting models such as [client/server](https://www.instaclustr.com/blog/cassandra-architecture/), [cloud](https://www.instaclustr.com/education/apache-cassandra-database/), [Local](https://www.instaclustr.com/education/apache-cassandra-database/) and [embedded](https://bookshelf.erwin.com/bookshelf/public_html/2021R2/Content/Release%20Notes/Feature%20Tour/Cassandra_Advanced_Denormalization.html).
